# Bento Dashboard Layout Plan — Borrower Home

## Problem

The current borrower Home desktop layout has:
- Max width capped at `max-w-6xl` (1152px), leaving unused horizontal space on wide screens
- Two-row layout with `lg:grid-cols-2` causing uneven card heights and large blank zones
- Repayment calendar detail panel squeezed to ~194px because the calendar card only occupies half the page width
- Inconsistent vertical rhythm with `gap-5` throughout

## Layout Design

Use an 8/4 column split on desktop. This creates a clean two-column bento with the left column holding larger/interactive cards and the right column holding compact metric cards.

### Desktop grid (12-col):

```
Row 1: [Next actions: col-span-8] [Financing overview: col-span-4]
Row 2: [Active loans: col-span-8] [Due this month: col-span-4]
Row 3: [Repayment calendar: col-span-8] [Profile readiness: col-span-4]
Row 4: [Payment progress: col-span-4, col-start-9]  (only if active loans exist)
```

This stacks as left-column (Next actions → Active loans → Repayment calendar) and right-column (Financing overview → Due this month → Profile readiness → Payment progress), aligned row-by-row.

### Mobile: all cards stack naturally via default grid behavior.
