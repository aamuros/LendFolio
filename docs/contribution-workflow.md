# Contribution Workflow

For commits to appear on a GitHub contribution graph, they must be reachable from the repository's default branch or the `gh-pages` branch. For LendFolio, the default branch is `main`, so direct documentation updates and merged pull requests should land on `main`.

After pushing a documentation-only change, confirm the commit appears on `main` before relying on it for contribution tracking.

Keep contribution-only documentation updates small and useful so they remain safe to commit directly to `main`.
